# Ludexa 2D Engine
Game engine for android 


Ludexa est un moteur de jeu 2D "No-Code" conçu pour fonctionner directement dans le navigateur. Il permet de créer des jeux et des prototypes sans écrire de code, grâce à un système d'édition visuelle par blocs (Blueprints).

## Fonctionnalités
*   **Éditeur visuel :** Interface intuitive avec glisser-déposer.
*   **Système de scripts :** Logique basée sur des nœuds (Blueprints).
*   **Multi-scènes :** Gestion de plusieurs scènes avec système de HUD.
*   **Sauvegarde locale :** Support du localStorage pour conserver la progression du joueur.
*   **Export Web :** Génération de builds HTML autonomes.

## Liste des fichiers
* **engine.js :** Moteur principal et boucle de jeu.
* **nodes-config.js :** Catalogue des blocs logiques (Blueprints).
* **nodes-interpreter.js :** Traducteur de la logique des scripts.
* **ui.js :** Gestion de l'interface utilisateur.
* **scenes.js :** Gestion des scènes et objets.
* **storage.js :** Import/Export de projets.
* **index.html :** Page principale de l'éditeur.
* **style.css :** Design de l'interface.
